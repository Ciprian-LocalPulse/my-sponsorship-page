# Code of Conduct

My Sponsorship Page is a public-facing sponsorship and trust-positioning repository with supporter, commercial, and public reputation context. All public interaction should remain professional, respectful, privacy-conscious, security-conscious, and legally calm.

## Expected Conduct

Participants are expected to:

- communicate respectfully and in good faith
- keep discussions focused on public documentation, governance, repository quality, and lawful sponsorship use cases
- avoid publishing sensitive sponsor, donor, payment, commercial, or private account details in public channels
- respect the Public Edition and private sponsorship-material boundary
- disclose security concerns through SECURITY.md rather than public issues
- use precise, professional language suitable for sponsors, supporters, commercial partners, procurement teams, and enterprise evaluators

## Unacceptable Conduct

The following behavior is not acceptable:

- harassment, personal attacks, discriminatory language, or intimidation
- requests for credential misuse, payment abuse, donor-data exposure, deceptive fundraising, impersonation, or bypass guidance
- publication of secrets, API keys, tokens, sponsor records, donor information, transaction details, private benefit details, private communications, or private implementation details
- attempts to extract private sponsorship data, private commercial terms, buyer records, payment records, unreleased brand materials, sponsor communications, analytics, or commercial implementation details
- spam, impersonation, misleading claims, or inflammatory language
- public vulnerability disclosure before coordination with the maintainer

## Security-Sensitive Communication

Do not post sensitive reports publicly. Use the process in SECURITY.md for vulnerability reports or repository safety concerns.

If a public issue contains sensitive information, the maintainer may edit, hide, lock, or remove it to protect sponsors, supporters, users, and the integrity of the sponsorship surface.

## Enforcement

The maintainer may take appropriate action to keep the repository safe and professional, including closing issues, declining pull requests, removing unsafe content, or restricting participation.

Enforcement decisions are made at the maintainer's discretion and are intended to preserve the lawful, enterprise-safe, sponsor-safe scope of the repository.

## Scope

This Code of Conduct applies to public repository interactions, including issues, pull requests, discussions, comments, and any public communication connected to My Sponsorship Page.
