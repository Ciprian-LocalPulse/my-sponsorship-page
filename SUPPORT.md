# Support

This repository is the Public Edition of My Sponsorship Page. Support is limited to public documentation, governance, repository hygiene, and safe evaluation questions.

## Public Support

For non-sensitive public questions, you may use GitHub Issues for:

- broken links
- documentation errors
- unclear public wording
- governance file corrections
- repository metadata or public-readiness feedback

Please do not include security-sensitive information, private sponsor records, donor information, payment details, transaction records, private commercial details, screenshots of private accounts, or sponsorship fulfillment assumptions in public issues.

## Sponsorship and Commercial Inquiries

For sponsorship, commercial collaboration, partnership discussion, advisory inquiries, or private supporter communication, contact the repository owner through the public GitHub profile or the contact path referenced in the repository README.

Commercial and sponsorship discussions should not be opened as public issues when they involve pricing, private benefits, payment records, donor information, partnership terms, fulfillment details, or confidential evaluation criteria.

## Security Reporting

Security issues must follow SECURITY.md.

Do not publish vulnerability details, secrets, tokens, donor data, sponsor records, payment details, private benefit details, commercial account information, or implementation-sensitive findings in public issues or discussions.

## Public Edition vs. Private Sponsorship Materials

Public support covers the Public Edition only. It does not include:

- private sponsorship account support through GitHub issues
- payment or transaction support in public channels
- private sponsor benefit fulfillment
- donor record review
- analytics or conversion strategy review
- private brand or campaign strategy review
- commercial partnership negotiation in public issues

Private sponsorship and commercial support, if available, requires an appropriate private channel, authorized sponsorship relationship, or separate written agreement.

## Response Expectations

This repository is maintained as a public sponsorship and governance surface. Public issue response times are best-effort and depend on severity, relevance, and maintainer availability.

Security reports, sponsorship inquiries, and commercial inquiries are prioritized through the appropriate private channels.
